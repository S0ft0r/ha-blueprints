**Версия 1**

- Добавлена базовая логика автоматического полива теплицы.
- Реализовано управление главным реле / клапаном подачи воды.
- Добавлен запуск полива по времени.
- Добавлен таймер автоматического отключения для защиты от перелива и перегрева оборудования.

**Версия 2**

- Добавлена поддержка нескольких клапанов направлений полива.
- Добавлена последовательная обработка направлений.
- Добавлены задержки открытия и закрытия клапанов.
- Добавлена поддержка разных типов исполнительных устройств: `switch` и `valve`.

**Версия 3**

- Добавлены отдельные настройки для утреннего и вечернего полива.
- Добавлена настройка длительности полива на каждое направление.
- Добавлены режимы работы:
    - каждый день;
    - через день;
    - выбор дней недели.
- Добавлена возможность использовать отдельные направления только утром или только вечером.
- Добавлена поддержка значений из вспомогательных сущностей `input_datetime` и `input_boolean`.

**Версия 4**

- Добавлена поддержка насоса.
- Добавлен предварительный запуск насоса перед началом полива.
- Добавлена задержка для набора давления в системе.
- Добавлена логика отключения насоса после завершения полива.
- Добавлен учет присутствия людей дома при отключении насоса.
- Добавлена обработка ручного включения направления полива.
- Добавлено автоматическое закрытие остальных направлений при ручном выборе одного направления.

**Версия 5**

- Добавлен учет прогноза погоды.
- Добавлена отмена полива при выбранных погодных условиях.
- Добавлена настройка глубины прогноза в часах.
- Добавлен автоматический расчет времени защитного таймера с учетом количества направлений.
- Добавлена поддержка нормально-открытых и нормально-закрытых клапанов направлений.
- Добавлены YAML-якоря для повторяющихся действий открытия и закрытия.
- Улучшена структура blueprint: настройки разделены на логические секции.
- Добавлена секция логирования ошибок.
- Добавлена запись ошибок в Logbook.
- Добавлена отправка уведомлений через выбранные `notify`-сущности.
- Начата интеграция диагностических проверок насоса и главного клапана.

# 🖼️ Dynamic Blinds & Battery Icon for Home Assistant

Этот сенсор генерирует динамическую SVG-иконку для рулонных штор. В одном изображении совмещены сразу два показателя: **реальное положение полотна шторы** и **уровень заряда её аккумулятора**.

## ✨ Основные особенности

*   **📏 Живая визуализация:** Высота шторы в иконке в точности соответствует её реальному положению (атрибут `current_position`).
*   **🔋 Индикатор батареи:** Справа от окна расположен вертикальный бар, который показывает уровень заряда.
*   **🎨 Цветовая индикация:** Цвет аккумулятора меняется автоматически:
    *   🟩 **Зеленый:** выше 50%
    *   🟨 **Желтый:** от 25% до 50%
    *   🟧 **Оранжевый:** от 10% до 25%
    *   🟥 **Красный:** ниже 10%
*   **🏗️ Чистый вектор:** Иконка создается на лету, всегда четкая и не требует внешних графических ресурсов.

## 🚀 Установка

Добавьте этот код в ваш `configuration.yaml` (раздел `template:`). Замените `cover.my_blinds` и `sensor.my_blinds_battery` на ваши реальные сущности.

```yaml
template:
  - sensor:
      - name: "Blinds Live Icon"
        state: "{{ state_attr('cover.my_blinds', 'current_position') | default(0) }}%"
        picture: >
          {% set pos = state_attr('cover.my_blinds', 'current_position') | int(0) %}
          {% set posb = states('sensor.my_blinds_battery') | int(0) %}
          
          {# Расчет высоты шторы #}
          {% set h = (1 - (pos / 100)) * 13.7 %}
          
          {# Расчет индикатора батареи #}
          {% set hb = ((posb / 100) * 18) | round(2) %}
          {% set yb = (21 - hb) | round(2) %}
          
          {# Логика цвета батареи #}
          {% if posb > 50 %}{% set b_col = "#4caf50" %}
          {% elif posb > 25 %}{% set b_col = "#fdd835" %}
          {% elif posb > 10 %}{% set b_col = "#ff9800" %}
          {% else %}{% set b_col = "#f44336" %}{% endif %}
          
          {% set color = "#44739e" %}
          
          {% set svg = "
          <svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 24 24'>
            <g transform='translate(2.4, 2.4) scale(0.8)'>
              <!-- Рама окна -->
              <path d='M 20,19 V 3 H 4 V 19 H 2 v 2 H 22 V 19 H 20 M 6,19 5.91,5.32 17.94,5.26 18,19 Z' fill='" ~ color ~ "'/>
              
              <!-- Полотно шторы -->
              <rect x='6' y='5.3' width='12' height='" ~ h ~ "' fill='" ~ color ~ "' opacity='0.5'/>
              
              <!-- Индикатор аккумулятора -->
              <rect x='22' y='3' width='2' height='15' fill='gray' opacity='0.2'/>
              <rect x='22' y='" ~ yb ~ "' width='2' height='" ~ hb ~ "' fill='" ~ b_col ~ "'/>
              
              <!-- Нижняя планка шторы (двигается вместе с полотном) -->
              <g transform='translate(0, " ~ h ~ ")'>
                <rect x='6' y='5.3' width='12' height='1.5' fill='" ~ color ~ "'/>
              </g>
            </g>
          </svg>
          " %}
          
          data:image/svg+xml,{{ svg | urlencode }}
```

## 🛠️ Как это работает

1.  **Геометрия шторы:** Полотно (`rect`) рассчитывается по формуле `(1 - (pos / 100))`. Это инвертирует значение: когда штора открыта на 100%, её визуальная высота в SVG становится равной 0.
2.  **Подвижная планка:** Нижняя утяжеляющая планка обернута в группу `<g>`, которая смещается по вертикали (`translate`) на ту же величину, что и высота шторы.
3.  **Батарея:** Высота цветного прямоугольника (`hb`) и его позиция по вертикали (`yb`) вычисляются динамически, чтобы полоска «росла» снизу вверх по мере заряда.

## 📝 Требования
*   Любая штора (cover) с поддержкой атрибута `current_position`.
*   Сенсор заряда батареи (0–100%).

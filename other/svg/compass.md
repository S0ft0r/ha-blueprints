# 🧭 Sun Compass & Room Orientation Icon

Этот сенсор визуализирует положение солнца на небосводе относительно плана вашего помещения (лоджии, комнаты или всего дома). Иконка представляет собой «живой» компас, который помогает понять, когда солнечные лучи начнут светить в конкретное окно.

## ✨ Основные особенности

*   **🏠 Процедурный план помещения:** Контур помещения генерируется динамически. Вы можете легко задать длину стен, расположение окон и их толщину.
*   **☀️ Азимут в реальном времени:** Иконка отображает реальное положение солнца, которое перемещается по орбите вокруг плана вашего дома.
*   **📐 Коррекция ориентации:** Параметр `rotate` позволяет повернуть план помещения так, чтобы он соответствовал реальному расположению дома относительно севера.
*   **🌙 Режим День/Ночь:** Цвет солнца меняется с ярко-желтого на серый при его уходе за горизонт (на основе `elevation`), а прозрачность снижается.
*   **🔍 Высокая точность:** Математически выверенные стыки стен и окон обеспечивают чистую векторную графику без разрывов.

```yaml
template:
  - sensor:
      - name: "Room Sun Compass"
        state: "{{ state_attr('sun.sun', 'azimuth') }}°"
        picture: >
          {# --- НАСТРОЙКИ ГЕОМЕТРИИ --- #}
          {% set height_long_wall = 12 %}
          {% set height_short_wall = 5 %}
          {% set rotate_room = 65 %}  {# Поворот дома относительно севера #}
          
          {# --- ДАННЫЕ СОЛНЦА --- #}
          {% set azimuth = state_attr('sun.sun', 'azimuth') | float(0) %}
          {% set elevation = state_attr('sun.sun', 'elevation') | float(0) %}

          {# (Ниже идет код генерации SVG из вашего шаблона) #}
```

## 🛠️ Настройка под себя

### 1. Изменение формы помещения
В массиве `sides` вы можете комбинировать стены (`wall`) и окна (`window`) в любом порядке. Параметр `h` отвечает за длину данного участка. По умолчанию настроен прямоугольник, где две стороны — глухие стены, а две другие — остекление.

### 2. Ориентация по сторонам света
Изменяйте переменную `set rotate = 65`. 
*   Установите `0`, если верхняя стена вашего плана смотрит строго на **Север**.
*   Меняйте значение, пока положение окон на иконке не совпадет с реальностью.

### 3. Габариты
*   `width_wall` (2.0): Толщина капитальных стен.
*   `width_window` (0.6): Толщина оконных рам.

## 📝 Техническая логика
Иконка использует SVG-трансформацию `rotate(azimuth, 12, 12)` для перемещения солнца по кругу. План помещения отрисовывается в центре холста 24x24 и поворачивается на фиксированный угол вашего дома, создавая персональную карту инсоляции.

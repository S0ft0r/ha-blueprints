# 💧 Animated Water Tank Level Icon for Home Assistant

Этот сенсор генерирует динамическую SVG-иконку для визуализации уровня жидкости в баке или резервуаре. Иконка снабжена плавной анимацией «живой воды» (волн), которая автоматически подстраивается под текущий объем заполнения.

## ✨ Основные особенности

*   **🌊 Анимированные волны:** Поверхность воды плавно колеблется с помощью встроенной SVG-анимации (SMIL), создавая эффект наполненности.
*   **📏 Точное заполнение:** Уровень воды в иконке линейно соответствует значению сенсора от 0% до 100%.
*   **🎨 Динамический цвет:** Цвет воды и корпуса бака можно менять «на лету» через вспомогательный текстовый элемент (`input_text`), что удобно для индикации состояний (например, красный цвет при критически низком уровне).
*   **🧊 Стеклянный блик:** На корпус наложен полупрозрачный слой, имитирующий отражение на поверхности бака для придания объема.
*   **⚡ Легкий код:** Никаких внешних изображений — вся графика и анимация вычисляются внутри Home Assistant.

## 🛠️ Настройка под себя

### 1. Связь с вашими данными
Замените `sensor.water_level` на ID вашего датчика уровня воды (0–100%).

### 2. Управление цветом
Сенсор ищет значение цвета в `input_text.tank_color`. 
*   Вы можете создать этот Helper в разделе «Вспомогательные элементы».
*   Это позволяет динамически менять цвет иконки (например, через автоматизацию: если уровень < 10%, установить значение `#f44336`).

### 3. Геометрия бака
Бак отрисован в системе координат 24x24. Область заполнения водой ограничена по высоте значениями от `6.2` до `18.2`. 

## 📝 Технические детали
Для анимации используется тег `<animate>`, который циклически переключает форму векторного пути между `wave_path_1` и `wave_path_2`, создавая эффект движения жидкости без нагрузки на процессор.
